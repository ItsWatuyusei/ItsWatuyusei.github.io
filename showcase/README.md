# Artisan Bakery Showcase

A premium product showcase built with modern web standards, adhering to the **ItsWatuyusei** architectural rules.

## Features
- **Modern UI**: Clean, premium design with HSL color palettes and Inter typography.
- **Dark Mode**: Automatic theme switching based on system preferences.
- **Search & Filter**: Real-time filtering by category (Sweet/Savory) and search query.
- **Multilingual Support**: Fully localized in English, Spanish, and Portuguese.
- **Performance**: Optimized with Shimmer skeleton loaders and lazy loading for images.

## Project Structure
- `product_showcase.html`: Semantic HTML5 structure.
- `styles.css`: CSS3 with design tokens and animations.
- `app.js`: ES6 module handling application logic.
- `config.js`: Centralized configuration (SSOT).

## Images
Generated using AI for a professional artisan bakery look.

Open `bakery_showcase/product_showcase.html` in any modern browser.
